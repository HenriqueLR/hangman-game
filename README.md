
##### Requisitos

	docker.io

	Chrome browser


##### Instalação

	make setup or ./setup.sh


##### Start Server

	make run_uwsgi or make run_gunicorn

	or

	./runserver.sh uwsgi or ./runserver.sh gunicorn


##### Colocando palavras no jogo

	Carrege um arquivo como de exemplo /env/db/words.csv

	make populate_db


##### Acessando o game

	http://localhost


##### Descrição

	Jogo da forca.
